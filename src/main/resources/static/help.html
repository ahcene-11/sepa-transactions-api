Référence complète des endpoints disponibles

URL Méthode Description de l'opération
/ GET Affiche la page d'accueil (HTML)
/help GET Affiche cette page de documentation (HTML)
/sepa26/resume/xml GET Affiche la liste simplifiée des 10 dernières transactions (XML)
/sepa26/resume/html GET Affiche la liste simplifiée des 10 dernières transactions (HTML)
/sepa26/xml/{id} GET Affiche le détail de la transaction {id} (XML)
/sepa26/html/{id} GET Affiche le détail de la transaction {id} (HTML)
/sepa26/insert POST Ajoute un flux SEPA complet via validation XSD (XML)
/sepa26/delete/{id} DELETE Supprime une transaction par son identifiant (XML)
